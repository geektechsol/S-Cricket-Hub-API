A comprehensive REST API for cricket scoring and management system.

Overview

This API provides complete functionality for cricket match management, including real-time scoring, player statistics, team management, and tournament organization.

Features

🔐 Authentication & Authorization

JWT-based authentication with role-based access control

👥 User Management

Complete user registration and profile management system

🏏 Teams & Players

Comprehensive team creation and player management with statistics

🏟️ Venues & Tournaments

Venue management and tournament organization with scheduling

🏏 Match Management

Live match scoring with real-time updates and ball-by-ball tracking

📊 Analytics & Reporting

Player performance analytics and detailed reporting tools

🔔 Notifications

Real-time notifications and event-based alerts

📁 File Management

Secure file upload and document management system

🔍 Search & Dashboard

Global search functionality and dashboard analytics

Basic Usage

Authentication

Most endpoints require JWT authentication. Include the token in the Authorization header:

Authorization: Bearer <your_jwt_token>

API Structure

The API follows REST conventions with proper HTTP methods:

  • GET - Retrieve resources and data
  • POST - Create new resources
  • PUT - Update existing resources
  • DELETE - Remove resources

Rate Limiting

  • General endpoints: 100 requests per minute
  • Authentication endpoints: 10 requests per minute
  • File uploads: 5 requests per minute

For detailed endpoint information and examples, please refer to the individual endpoint documentation below.
